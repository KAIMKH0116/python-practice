# 2025年5月19日 学習ログ

## 学んだこと
- リスト + ループ + 計算の基本パターン
- 問題を解くための「4ステップ思考法」
- if文を使った条件判定（最大値の見つけ方）

## 4ステップ思考法
1. 問題文を3行で言い直す
2. 使う文法を決める
3. 大まかな骨組みを言葉で書く
4. コードに変える

## 実践した問題
1. 4人の子どものお小遣い管理（合計・平均）
2. 7日間の勉強時間管理（合計・平均）
3. 5件のフリーランス報酬管理（合計・平均）
4. 4人のテスト点数から最大値を見つける

## 重要な学び
- 「ゼロから書き始める時に手が止まる」→ パターン認識で解決
- ループの中で条件判定：`if tensuu[i] > saikouten:`
- 変数を更新する：`saikouten = tensuu[i]`
